Dijkstra's Algorithm Solver

By Mostafa Dahshan

Usage

  • While Draw node is selected, click anywhere in the canvas to create a node.
  • To draw a link between two nodes, select the Draw link radio button, then click on the nodes you want to connect.
  • To change the cost or node label, click on the cost or the label while Set cost or label radio button is selected.
  • Finally, to run the algorithm, select Set Start then click on the starting node.
